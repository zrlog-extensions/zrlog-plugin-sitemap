# zrlog-plugin-sitemap

ZrLog 站点地图插件。生成 `sitemap.xml`，并可在站点侧边栏显示站点地图入口。

## 功能

- 配置 sitemap 文件路径
- 配置站点地图入口说明文字
- 按文章列表生成 sitemap 文件
- 可将站点地图文件同步到静态资源存储

## Sitemap 协议

https://www.sitemaps.org/protocol.html

## 构建

```shell
export JAVA_HOME=${HOME}/dev/graalvm-jdk-latest
export PATH=${JAVA_HOME}/bin:$PATH
```
